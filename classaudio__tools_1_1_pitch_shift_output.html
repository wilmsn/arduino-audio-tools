Detailed Description

template<typename T, class BufferT>
class audio_tools::PitchShiftOutput< T, BufferT >

Pitch Shift: Shifts the frequency up or down w/o impacting the length! We reduce the channels to 1 to calculate the pitch shift and provides the pitch shifted result in the correct number of channels. The pitch shifting is done with the help of a buffer that can have potentially multiple implementations.
